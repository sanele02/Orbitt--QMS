Security

Your data security is our top priority. Learn about our comprehensive security measures.

Data Protection

End-to-End Encryption

All data transmitted between your browser and our servers is encrypted using industry-standard TLS 1.3 encryption.

Database Security

Your data is stored in Google Firebase with enterprise-grade security, including encryption at rest and in transit.

Authentication & Access

Secure Authentication

Firebase Authentication with email/password verification and secure session management.

Access Controls

Role-based access controls ensure only authorized users can access your queue data.

Infrastructure Security

Google Cloud Platform

Hosted on Google's secure infrastructure with 99.9% uptime SLA and automatic security updates.

HTTPS Only

All connections are forced through HTTPS with HSTS headers for maximum security.

Continuous monitoring for security threats and suspicious activities.
